# 物語論の語彙集 — 構成設計の共通言語

[00-core.md](./00-core.md) の理論的土台となる辞書です。物語論（Narratology）の用語を、**日本語定訳・一行説明・ブログ／発表資料／小説への転用例**の形でまとめます。Skill・SubAgent はこの語彙を共通言語として診断・提案を行います。

> 学術的な厳密さより「構成を語るときに名前を呼べること」を優先しています。原典に当たりたいときは [Narratology (en)](https://en.wikipedia.org/wiki/Narratology) / [物語論 (ja)](https://ja.wikipedia.org/wiki/物語論)。

---

## 1. 基本区別 — 何を語るか / どう語るか

物語を「素材」と「提示」に分ける、最も基本的で実用的な区別。

| 用語（原語）                  | 日本語定訳   | 説明                                       | 転用例                                               |
| :---------------------------- | :----------- | :----------------------------------------- | :--------------------------------------------------- |
| fabula / histoire             | 物語内容     | 時系列に並んだ出来事・事実そのもの         | 起きたことを全部書き出した素材リスト                 |
| syuzhet / récit               | 物語言説     | それを受け手にどう並べ・どう見せるか        | 結論先出し・伏線・要約と詳述の配分                    |
| narration                     | 語り         | 「語る」という行為そのもの                  | 誰が・どの立場から語っているか                       |

→ 構成作業＝ fabula を素材に、最適な syuzhet を設計すること。[00-core.md](./00-core.md) の (B) を参照。

## 2. ジュネットの時間論 — 緩急のコントロール

物語言説の「時間の扱い」を 3 軸で分解する。

### 順序（order）

物語内容の時間順と、提示順のズレ。

| 用語         | 日本語定訳         | 説明                       | 転用例                               |
| :----------- | :----------------- | :------------------------- | :----------------------------------- |
| analepsis    | 後説法（回想）     | 過去に遡って語る           | 伏線回収、ブログの「そもそもの経緯」 |
| prolepsis    | 先説法（予示）     | 先の出来事を先取りして語る | 結論先出し、PREP の結論、引きの一文   |

### 持続 / 速度（duration）

物語内容の長さと、叙述の長さの比。**どこを厚く、どこを薄くするか**。

| 用語     | 日本語定訳 | 説明                                 | 転用例                       |
| :------- | :--------- | :----------------------------------- | :--------------------------- |
| pause    | 休止法     | 物語内時間は進まず描写だけ進む       | 風景・心情の描き込み、図の解説 |
| stretch  | 伸長法     | 叙述が実時間より長い（スロー）       | クライマックスの細密描写     |
| scene    | 情景法     | 物語内時間と叙述がほぼ等速           | 会話シーン、デモ             |
| summary  | 要約法     | 長い時間を短く圧縮                   | 「数か月後」「背景を一言で」 |
| ellipsis | 省略法     | 出来事を飛ばす                       | 重要でない期間のカット       |

→ **ハイライトに速度を寄せる**（情景・伸長）、周辺は要約・省略で削る。[00-core.md](./00-core.md) 観点 4 / 7。

### 頻度（frequency）

同じ出来事を何回語るか。

| 用語        | 日本語定訳 | 説明                       | 転用例                       |
| :---------- | :--------- | :------------------------- | :--------------------------- |
| singulative | 単起法     | 1 回起きたことを 1 回語る  | 通常の叙述                   |
| repetitive  | 反復法     | 1 回の出来事を複数回語る   | 核フレーズのリフレイン、視点替え |
| iterative   | 括復法     | 反復した出来事をまとめて語る | 「毎朝こうしていた」         |

## 3. 焦点化（focalization） — 視点

「誰の知覚を通して語るか」。語り手（誰が語るか）とは別の軸。

| 用語             | 日本語定訳     | 説明                                       | 転用例                             |
| :--------------- | :------------- | :----------------------------------------- | :--------------------------------- |
| zero focalization| 焦点化ゼロ     | 全知。あらゆる内面・時空を把握             | 神視点の小説、俯瞰の解説記事       |
| internal         | 内的焦点化     | ある人物の知覚した範囲だけを描く           | 一人称的な語り、体験談ブログ       |
| external         | 外的焦点化     | 外面のみ。内面は描かない                   | ルポ、客観記述                     |

→ 一貫性が命。無自覚な揺れは [00-core.md](./00-core.md) 観点 5 で警告。

---

## 4. 語り手の類型 — 誰が語るか

| 用語                 | 日本語定訳         | 説明                                       | 転用例                   |
| :------------------- | :----------------- | :----------------------------------------- | :----------------------- |
| homodiegetic         | 等質物語世界的     | 語り手が作中人物（「私」と言える）         | 一人称小説、体験談       |
| heterodiegetic       | 異質物語世界的     | 語り手は作中人物でない                     | 三人称小説、解説記事     |
| extradiegetic        | 物語世界外的       | 物語の外側にいる第一次の語り手             | 地の文の語り手           |
| intradiegetic        | 物語世界内的       | 作中で別の話を語る人物（入れ子）           | 作中作、引用された逸話   |

## 5. 役割構造 — 登場要素を機能で捉える

「キャラ／要素」を**機能（はたらき）**で抽象化する。小説以外でも、登場する概念・人・ツールの役割整理に使える。

### グレマスのアクタントモデル

3 対 6 項の関係で物語の力学を表す。

| アクタント       | はたらき                       | ブログ／発表での転用例                 |
| :--------------- | :----------------------------- | :------------------------------------- |
| 主体 / 対象      | 行為者と、その目標             | 読者（主体）／解決したい課題（対象）   |
| 送り手 / 受け手  | 目標を与える側／受け取る側     | きっかけ（送り手）／受益者（受け手）   |
| 助力者 / 敵対者  | 助けるもの／妨げるもの         | 紹介するツール（助力者）／障害（敵対者）|

### プロップの形態学

- **31 の物語機能** — 昔話に共通して現れる行為の型（禁止・違反・出発・試練・帰還…）。**順序がほぼ不変**であることが「構成の型」の理論的根拠。
- **7 つの行動領域（役割）** — 悪者／贈与者／助手／主人公／偽の主人公／王女（と父）／派遣者。

## 6. 筋立て（アリストテレス『詩学』）

| 用語（原語）   | 日本語定訳     | 説明                                       | 転用例                           |
| :------------- | :------------- | :----------------------------------------- | :------------------------------- |
| mythos         | 筋（ミュトス） | 出来事の因果的な配置                       | 構成そのもの                     |
| peripeteia     | 逆転           | 状況が反対方向へ急転                       | どんでん返し、「実は逆だった」    |
| anagnorisis    | 認知           | 無知から知への転換・発見                   | 気づき、「答え合わせ」            |
| catharsis      | カタルシス     | 受け手の感情の浄化・解放                   | 読後感、締めの余韻               |

→ ペリペテイア（逆転）とアナグノリシス（認知）が同時に来ると最も強い、とアリストテレスは言う。

## 7. 不変条件 — どの物語にも共通する普遍構造

[00-core.md](./00-core.md) の (A) の理論的出典。

- **トドロフの均衡モデル**: 均衡 → 不均衡 → 新たな均衡。
- **始め・中・終わり**: 因果でつながった 3 部構造。
- **物語性（narrativity）**: 受け手が「物語として経験・意味化できる」質（M.-L. Ryan）。→ 受容・認知の側からの深掘りは [02-cognitive.md](./02-cognitive.md)。
- **story / discourse の不可分性**: 内容と提示は分けて設計できるが、最終的には一体として受け取られる。→ この「分けて設計できるが一体で受け取られる」の力学を 2 軸（生成スタック / 受容の対）で整理したのが [02-cognitive.md](./02-cognitive.md) §1。
