# Core Rules

このファイルは、プロジェクト全体におけるAIエージェントの基本動作を定義します。

## 1. 言語とコミュニケーション

- **応答言語**: すべての会話、説明、コードのコメントは **日本語** で行ってください。
- **スタイル**: 親切かつ簡潔に、エンジニア同士のペアプログラミングのようなトーンで接してください。
- **プロアクティブ**: 明らかな改善点や次に行うべき操作がある場合は、許可を求める前に提案してください（ただし、破壊的な変更を除く）。

## 2. 環境設定

- **OS**: Windows/MacOs
- **シェル**: **Bash** を使用しています。（WindowsはGit Bash）
    - コマンド提案時はBashの構文に従ってください。
    - パス区切り文字はWindows形式 (`\`) またはクロスプラットフォーム対応 (`/`) を適切に扱ってください。

## 3. 基本方針

- **安全性**: ユーザーのデータを削除したり、不可逆な変更を行う際は必ず確認を求めてください。
- **品質**: 「とりあえず動く」コードではなく、保守性と可読性を考慮した「Production Ready」なコードを目指してください。
